## Component Contracts (for subagent reference)

### OnlineConsultation
- Server component (no client interactivity needed)
- Uses `SectionWrapper` for fade-in
- ₹499 + 4 check-bullet features + WhatsApp CTA
- Centered max-w-[640px] card on `bg-primary/5` strip

### ServicesCarousel
- Client component (`"use client"`)
- Embla carousel, loop, 3 visible peek-center-peek desktop / 1 mobile
- Center card autoplays video muted, sides paused with poster
- Click side card → scrollTo center
- Volume toggle on active card (Volume2/VolumeX)
- WhatsApp CTA in active overlay
- prefers-reduced-motion guards
- aria-roledescription="carousel", aria-current on dots

### WhyChooseUs
- Server component
- 5 cards, lg:grid-cols-5, sm:grid-cols-2, grid-cols-1
- Reads from `data/why-choose-us.ts`
- Circular icon badge + title + 1-line subtitle
- Hover lift effect

### VideoTestimonials
- Client component (modal state)
- CSS columns masonry: lg:columns-3, sm:columns-2, columns-1
- Each tile: poster + Play overlay + name pill
- Click → opens shadcn Dialog with autoplay video
- Esc/backdrop closes
- prefers-reduced-motion: don't autoplay in modal

### Testimonials (rewritten)
- Server component
- Static 3-card grid (no carousel, no autoplay)
- Reuse existing card design (avatar initial, stars, quote, name)
- Section header with subtitle: "Hear from people we've helped recover and thrive."

### Footer (rewritten)
- Server component
- 4 columns: Brand+Socials | Our Policies | Know Us | Contact
- Add X, LinkedIn, Medial inline SVG social icons
- Phone: tel link, Email: mailto link
- Bottom bar unchanged
